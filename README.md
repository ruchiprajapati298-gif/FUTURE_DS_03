# 🎯 Marketing Funnel & Conversion Performance Analysis

## 📝 Overview
This repository contains an end-to-end analysis of a simulated digital marketing funnel. The goal of this project is to track the customer journey from a website visitor, to a captured lead, and finally into a paying customer. 

By analyzing the specific conversion rates across multiple digital channels (Organic Search, Paid Ads, Social Media, Email), this project highlights where users drop off, measures raw channel efficiency against Customer Acquisition Cost (CAC), and uses time-series anomaly detection to pinpoint broken funnel loops. 

**Key Objectives Assessed:**
- 📉 Macro Funnel Conversion Rates
- 💸 Channel Efficiency & CAC Optimization 
- ⏱️ Time-Series Anomaly Tracking
- 💡 Marketing Spend Resource Allocation
